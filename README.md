# Demographic Update Data Analysis

This project is based on Aadhaar Demographic update data.
The analysis is done using Python and visualized using Power BI.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Power BI

## What is in this project
- State-wise Demographic update analysis
- District-wise Demographic update analysis
- Age-wise Demographic update analysis
- Date-wise trend analysis

## Purpose
This project is created for learning data analysis and visualization.
